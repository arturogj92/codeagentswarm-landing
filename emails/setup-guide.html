Setup de Email Autenticado

Para que los emails lleguen a Primary y no a Promotions/Spam

1
Registrar dominio en Resend
  1. Abre Resend y ve a Domains
  2. Click "Add Domain"
  3. Escribe: codeagentswarm.com
  4. Region: us-east-1
  5. Click "Add"
  6. NO CIERRES la pantalla - necesitas los DNS records para el paso 2
Abrir Resend Domains →
2
Anadir DNS Records en Namecheap

Abre Namecheap, ve a tu dominio y a la pestana "Advanced DNS". Anade estos 5 records:

Abrir Namecheap Advanced DNS →

Record 1: SPF (TXT)

CampoValor
TypeTXT Record
Host@
Valuev=spf1 include:send.resend.com ~all
TTLAutomatic
Importante: Si ya tienes un TXT record que empieza por v=spf1, NO crees otro. Edita el existente y anade include:send.resend.com antes de ~all. Solo puede haber UN record SPF.

Records 2, 3, 4: DKIM (3x CNAME)

Copia los 3 CNAMEs exactos desde la pantalla de Resend. Seran algo asi:

TypeHostValue (copiar de Resend)
CNAME resend._domainkey copiar de Resend dashboard
CNAME resend2._domainkey copiar de Resend dashboard
CNAME resend3._domainkey copiar de Resend dashboard
Los valores exactos de los CNAME son unicos para tu dominio. Copialos tal cual desde la pantalla de Resend (parecen algo como p3tre...dkim.resend.dev).

Record 5: DMARC (TXT)

CampoValor
TypeTXT Record
Host_dmarc
Valuev=DMARC1; p=none; rua=mailto:hello@codeagentswarm.com
TTLAutomatic

Checklist - marca cada record que hayas anadido:

3
Verificar dominio en Resend
  1. Vuelve a Resend > Domains
  2. Click "Verify DNS Records" en tu dominio
  3. Espera 1-5 minutos (Namecheap suele ser rapido)
  4. Recarga la pagina hasta que todos los records muestren checkmark verde
Si algun record no verifica despues de 10 minutos, revisa que los valores CNAME sean exactos (sin espacios extra ni puntos al final).
Abrir Resend Domains → Verificar DNS en MXToolbox →
4
Generar API Key (si no la tienes)

Si ya tienes una API key de Resend (la re_YSR... que usamos), puedes saltar este paso.

  1. En Resend, ve a API Keys
  2. Click "Create API Key"
  3. Nombre: codeagentswarm-emails
  4. Permission: Sending access
  5. Domain: codeagentswarm.com
  6. Copia la key (empieza por re_)
Abrir Resend API Keys →
🎉

Setup completado

Tu dominio esta autenticado con SPF + DKIM + DMARC.
Los emails ahora llegaran a Primary en vez de Promotions.
